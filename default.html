Metalink is an Internet standard that harnesses the speed and power of peer to peer networking and traditional downloads with a single click.

For users and content providers, Metalink makes downloading simpler, faster, and more reliable.

Popular Metalinks: LibreOffice, openSUSE DVD / CD, Ubuntu, Sage, Mandriva, Fedora, Sabayon, OpenOffice.org

Want to get involved? Join our open source project!

  • If one link or server fails, download automatically continues using another.
  • One Click Downloads. Easy user experience.
  • Minimizes and distributes traffic so individual servers are under less strain.
  • Downloads are verified for enhanced reliability.
  • Structured information about downloads.
  • Increased security.
  • Extremely fast download speeds.
  • Automatic error recovery and repair of corrupted downloads.
  • Combines FTP and HTTP with optional Peer-to-peer (P2P, shared bandwidth).
  • Easily add multiple files to a download queue.
  • No Single Point of Failure (SPOF) like FTP or HTTP URLs. More fault tolerant.
  • Allows sites to publish where resources are duplicated on a mirror network/CDN to improve caching.
  • Transparent usage where people can get the benefits of Metalink without even knowing they're using it.
